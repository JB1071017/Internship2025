How It Works

1. Packet Capture

Uses PyShark to capture network packets without requiring elevated privileges.

2. Feature Extraction

Extracts key metrics like flow duration, packet sizes, and transmission rates.

3. ML Analysis

Pre-trained model classifies traffic as benign or specific threat types.

4. Alerting

Immediate notifications via email when threats are detected.
